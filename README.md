# Grant Writing Assistant

**Stop leaving grants on the table. Get submission-ready applications in hours, not weeks.**

Grant consultants charge $5,000–$15,000 per proposal. This skill gives you the same structured approach for $29 — four prompts that take you from opportunity research to a complete, reviewable application.

---

## What This Skill Does

Four prompts. Complete grant application.

| Prompt | What It Produces |
|--------|-----------------|
| 1. Grant Opportunity Finder | Top 10 opportunities ranked by fit score, quick-win pipeline, red flags to avoid |
| 2. Executive Summary & Needs Statement | The opening sections reviewers weight most heavily — 250-word summary + 400-word needs statement |
| 3. Project Narrative & Work Plan | Program description, evidence base, month-by-month timeline, logic model |
| 4. Evaluation Framework & Budget Justification | Indicators table, budget narrative with justifications, cost-effectiveness calculation |

## What You Need to Start

**Prompt 1 (Opportunity Finder):** Mission, program description, budget range, location, org type, target population, sector.

**Prompts 2-4 (Application Writing):** A specific grant you're applying for, plus program details (goals, activities, staff, timeline, budget).

Best workflow: Run Prompt 1 first to identify the right grant, then run Prompts 2-4 for that specific application.
